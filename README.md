> Prueba de módulo
- Pizzería Mamma Mía!

## Objetivo
- /Home: Mostrar el catálogo de pizzas.
- /pizza/:id: Mostrar el detalle de una pizza seleccionada en el catálogo.
- /carrito: Mostrar las pizzas añadidas al carrito con sus precios y total a pagar.
